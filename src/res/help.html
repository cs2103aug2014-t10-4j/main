Dear Doublist, thank you for using DoubleUp to organize your to-dos!

Below is a list of all the things that you can do in DoubleUp.


IF YOU WANT TO TYPE LIKE THIS SHORTHAND

Basic actions
add a task add 31 Dec assignment .a 31 Dec assignment
mark a task as completed delete 1 .d 1
change your task edit 1 this is a new name .e 1 this is a new name
find tasks with a keyword search assignment .s assignment
see everything in your list show all .sha
sort your list chronologically sort time .st

Advanced actions
clear your archive clear archive .ca
delete all your tasks delete all .da
delete all today tasks delete today .dtd
delete all past tasks delete past .dp
see the help screen help .h
hide additional details hide details .hd
set importance level for a task add Exam importance 3 .a Exam .i 3
see all your floating tasks show floating .shf
see all your tasks for today show today  .sht
see all your tasks for the week show week .shw
see all your tasks for next 7 days show this week .stw
see additional details show details .sd
sort your list alphabetically sort alpha .sa
sort your list by importance sort importance .si
view all your completed tasks view archive .va
redo your last action redo .r
undo your last action undo .u

*Tip: To remember all the shorthand commands easily, note that they are formed with a dot followed by the first letters of the first few consonants of the full command.


More Keyboard Shortcuts for Windows users

CTRL + SPACE To maximise and minimise DoubleUp from anywhere.
F2 To see the help screen.
ESC To go back to all tasks.
Up/Down To select previous inputs.
